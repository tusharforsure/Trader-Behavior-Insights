# Trader Behavior Insights – Fear & Greed Analysis

## Objective
This project analyzes the relationship between trader performance and Bitcoin market sentiment using Hyperliquid trading data and the Bitcoin Fear & Greed Index.

## Datasets
- Historical Trader Data (Hyperliquid)
- Bitcoin Fear & Greed Index

> Raw datasets are not included due to size and licensing constraints.

## Key Analysis
- Trade performance vs market sentiment
- Win/Loss behavior across Fear and Greed phases
- Risk and volatility comparison
- Strategy-driven insights for smarter trading

## Key Insights
- Trading activity and position size increase during Greed phases
- Higher volatility and downside risk observed during Extreme Greed
- Fear phases show more consistent trader performance

## Files
- `trader_behavior_analysis.ipynb` – Complete analysis notebook
- `requirements.txt` – Python dependencies
